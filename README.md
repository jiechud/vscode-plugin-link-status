### Npm & Yarn Link List
* npm和yarn link的列表， 支持普通项目和`lerna`项目
* 支持unlink操作


### 使用教程
* 该插件会列出项目当中已link的包，在你需要link包的地方执行`yarn link`。在link列表会看到这些包
* 可以方便的手动添加link和删除link
